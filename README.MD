# ZZC

A (currently incomplete) reimplementation of ZZT from scratch, 

started by EGAMatsu (Digi-Space Productions, aka Luke Warren) in late 2025; the goal of this project is to be a clean reimplentation of the Engine in C for porting.

## Current status
- Level loading implemented (using libzzt2)
- (Partial) Implementation of core mechanics
- (Partial) Support for non Code Pade 437 Character sets
